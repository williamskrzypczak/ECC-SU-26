Chapter 5 Lab: The Art of Displaying Grids

Introduction

You will build a campus course catalog displayed in grid layouts that scroll vertically and horizontally.

Lists work well for rows of text. Grids work better when you want a bulletin board or photo wall look. This chapter teaches you how to arrange items in rows and columns.

You can complete this lab in about 15 minutes.

Learning Objectives

After completing this lab, you will be able to:

  • Build a simple static Grid with GridRow
  • Create a scrollable LazyVGrid for vertical grid layouts
  • Configure columns with GridItem (.fixed, .flexible, .adaptive)
  • Build a horizontal LazyHGrid inside a horizontal ScrollView
  • Apply conditional formatting to highlight selected grid items
  • Explain how adaptive grids respond to different screen sizes

Key Ideas

Grid is like a small spreadsheet with labeled cells. It is great for simple, fixed layouts.

LazyVGrid is like a vending machine that only loads the snacks you can see. GridItem sets the shelf rules: fixed width, flexible sharing, or adaptive fitting.

grid concepts and simple analogies
Concept What it does Analogy
Grid + GridRow Static rows and columns for small layouts A spreadsheet with a few labeled cells
LazyVGrid Vertical grid that loads cells as they appear A vending machine stocking rows top to bottom
LazyHGrid Horizontal grid for side-scrolling content A film strip you pull left and right
GridItem(.fixed) Column or row with an exact size A locker with one fixed width
GridItem(.flexible) Column or row that shares leftover space Stretchy shelves that expand to fill a wall
GridItem(.adaptive) Fits as many items as possible per row Water filling a tray and taking whatever space fits
Conditional formatting Changes style based on state or data Highlighting a chosen card on a bulletin board

Lab Scenario: Campus Course Catalog Grid

You will display campus courses in grid layouts. Students browse courses in a vertical grid, swipe through featured courses horizontally, and tap to highlight a favorite.

Lab Instructions

Part 1: Build a Simple Static Grid

Topics: The Grid view and Creating a simple static grid.

Start with a small Grid for a fixed schedule preview. This is not scrollable. It works well for a few rows.

import SwiftUI

struct SchedulePreview: View {
    var body: some View {
        Grid(alignment: .leading, horizontalSpacing: 16, verticalSpacing: 8) {
            GridRow {
                Text("Course")
                    .fontWeight(.bold)
                Text("Room")
                    .fontWeight(.bold)
            }
            GridRow {
                Text("SwiftUI I")
                Text("204")
            }
            GridRow {
                Text("SwiftUI II")
                Text("210")
            }
        }
        .padding()
        .background(Color(.systemGray6))
        .cornerRadius(10)
    }
}

Part 2: Create Course Data and a Card View

Topics: Displaying grids in iOS.

Define course data and a card view you will reuse in lazy grids.

struct Course: Identifiable {
    let id = UUID()
    let code: String
    let title: String
    let isFeatured: Bool
}

let courses: [Course] = [
    Course(code: "MAD 157", title: "SwiftUI Basics", isFeatured: true),
    Course(code: "MAD 158", title: "iOS Data", isFeatured: true),
    Course(code: "MAD 159", title: "App Design", isFeatured: false),
    Course(code: "MAD 160", title: "Core Data", isFeatured: false),
    Course(code: "MAD 161", title: "Networking", isFeatured: true),
    Course(code: "MAD 162", title: "Animations", isFeatured: false),
    Course(code: "MAD 163", title: "Testing", isFeatured: false),
    Course(code: "MAD 164", title: "App Store", isFeatured: false)
]

struct CourseCard: View {
    let course: Course
    let isSelected: Bool

    var body: some View {
        VStack(alignment: .leading, spacing: 6) {
            Text(course.code)
                .font(.caption)
                .fontWeight(.bold)
            Text(course.title)
                .font(.subheadline)
                .lineLimit(2)
        }
        .frame(maxWidth: .infinity, minHeight: 80, alignment: .leading)
        .padding()
        .background(isSelected ? Color.blue.opacity(0.2) : Color(.systemGray6))
        .overlay(
            RoundedRectangle(cornerRadius: 10)
                .stroke(isSelected ? Color.blue : Color.clear, lineWidth: 2)
        )
        .cornerRadius(10)
    }
}

Part 3: Build a LazyVGrid with GridItem

Topics: Lazy grids and Using GridItem to control layout.

Use three flexible columns for a balanced vertical grid. Wrap it in a ScrollView so users can scroll through many courses.

let threeColumns = [
    GridItem(.flexible()),
    GridItem(.flexible()),
    GridItem(.flexible())
]

ScrollView {
    LazyVGrid(columns: threeColumns, spacing: 12) {
        ForEach(courses) { course in
            CourseCard(course: course, isSelected: false)
        }
    }
    .padding()
}

Try changing one column to GridItem(.fixed(120)). Notice how the first column stays the same width while others flex.

Part 4: Use an Adaptive GridItem

Topics: Scrolling horizontally and sizing the grid automatically, and Reacting to device rotation.

Replace your column array with an adaptive layout. SwiftUI fits as many cards per row as the screen allows.

let adaptiveColumns = [
    GridItem(.adaptive(minimum: 140), spacing: 12)
]

Preview on iPhone and iPad sizes. Rotate the simulator. The number of columns should change automatically. This is how grids react to device size.

Part 5: Horizontal Featured Courses with LazyHGrid

Topics: Lazy grids and horizontal scrolling.

Show featured courses in a horizontal film strip above the main grid.

let featuredRows = [GridItem(.fixed(100))]

ScrollView(.horizontal, showsIndicators: false) {
    LazyHGrid(rows: featuredRows, spacing: 12) {
        ForEach(courses.filter { $0.isFeatured }) { course in
            VStack {
                Image(systemName: "star.fill")
                    .foregroundColor(.orange)
                Text(course.code)
                    .font(.caption)
            }
            .frame(width: 100, height: 100)
            .background(Color(.systemGray6))
            .cornerRadius(10)
        }
    }
    .padding(.horizontal)
}

Part 6: Conditional Formatting with Selection

Topics: Conditional formatting of a view.

Let students tap a course card to highlight it. Use @State to track the selected course.

struct ContentView: View {
    @State private var selectedCourseID: UUID?

    var body: some View {
        NavigationStack {
            VStack(spacing: 16) {
                Text("Featured Courses")
                    .font(.headline)
                    .frame(maxWidth: .infinity, alignment: .leading)
                    .padding(.horizontal)

                // Horizontal featured strip from Part 5 goes here

                Text("All Courses")
                    .font(.headline)
                    .frame(maxWidth: .infinity, alignment: .leading)
                    .padding(.horizontal)

                ScrollView {
                    LazyVGrid(columns: adaptiveColumns, spacing: 12) {
                        ForEach(courses) { course in
                            CourseCard(
                                course: course,
                                isSelected: selectedCourseID == course.id
                            )
                            .onTapGesture {
                                selectedCourseID = course.id
                            }
                        }
                    }
                    .padding()
                }
            }
            .navigationTitle("Course Catalog")
        }
    }
}

The card border and background change when selected. This is conditional formatting based on state.

Troubleshooting

Grid Items Stack in One Column

Check your GridItem array. Each item defines one column in a LazyVGrid. Use .adaptive(minimum:) with one GridItem to auto-fit many columns.

LazyVGrid Does Not Scroll

Wrap the grid in a ScrollView. The grid arranges items but does not scroll by itself.

LazyHGrid Shows One Long Row Off Screen

Put LazyHGrid inside ScrollView(.horizontal). Define rows with GridItem, not columns.

Selection Highlight Does Not Appear

Confirm selectedCourseID == course.id is passed to CourseCard. Tap handlers must update selectedCourseID inside the onTapGesture closure.
