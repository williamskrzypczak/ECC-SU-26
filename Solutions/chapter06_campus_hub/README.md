# Chapter 6 Solution: Campus Hub

## Setup

1. Create a new App project named `CampusHub`.
2. Add all tab view files plus `ContentView.swift`.

## What You Should See

- Three tabs: Home, Clubs, Profile
- RSVP sheet slides up from the Home tab
- Leave club alert on the Clubs tab
- Help popover on the Profile tab

## Files

- `ContentView.swift` — root TabView
- `HomeTab.swift` — sheet trigger
- `ClubsTab.swift` — confirmation alert
- `ProfileTab.swift` — help popover
- `RSVPSheet.swift` — modal with dismiss
