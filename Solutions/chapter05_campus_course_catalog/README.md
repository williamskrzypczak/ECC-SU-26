# Chapter 5 Solution: Campus Course Catalog

## Setup

1. Create a new App project named `CampusCatalog`.
2. Add `Course.swift`, `CourseCard.swift`, and `ContentView.swift`.

## What You Should See

- Static schedule grid at the top
- Horizontal featured course strip with star icons
- Adaptive course grid that reflows on wider screens
- Blue highlight when a course card is tapped

## Files

- `Course.swift` — model, sample data, grid column definitions
- `CourseCard.swift` — selectable grid cell
- `ContentView.swift` — schedule preview, LazyHGrid, LazyVGrid
