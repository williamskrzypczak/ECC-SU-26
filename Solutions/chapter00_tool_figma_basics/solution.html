This reference shows the expected wireframe for the Chapter 0 Figma lab. There is no code to copy. Your finished Figma file is the solution.

Layer Checklist

Your completed file should include these layers:

  • File: Campus Welcome Wireframe
  • Frame: Welcome Screen (iPhone 15 size)
  • Frame: Detail Screen (duplicate of Welcome Screen)
  • Text: Campus Welcome (title on Welcome Screen)
  • Text: Built by Your Name (subtitle)
  • Text: Learning SwiftUI one view at a time (tagline)
  • Component: PrimaryButton with label Get Started
  • Auto layout group wrapping title, subtitle, tagline, and button
  • Prototype link: Get Started → Detail Screen

Sample Reflection Answers

Frame vs. group: A frame has a fixed size like a phone screen. A group just bundles layers together without a device size.

Auto layout: Auto layout stacks and spaces layers automatically so you do not position every item by hand.

Components: A component lets you update every button at once by editing the master design.

Troubleshooting Quick Reference

Common problems and fixes
Problem Fix
Frame tool missing Press F on the keyboard
Auto layout wrong direction Set direction to Vertical in properties
Prototype will not play Check blue arrow exists, then click Present
Export is blank Select the outer frame, not an inner layer
Component not created Select both rectangle and label before Cmd+Option+K
