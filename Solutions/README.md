# MAD 157 Lab Solutions

Reference solutions for the SwiftUI labs. Each chapter folder includes:

- `solution.html` — browse on GitHub Pages (setup steps + full code)
- `README.md` — quick reference in the repo
- `.swift` files — copy into Xcode

Open `Solutions/index.html` in a browser or use the Lab Solutions table on the main `index.html` page.

## Notes for Instructors

- Chapters 14 and 15 include `SAMPLE_ANSWERS.md` for reflection question portions.
- Each folder contains a `README.md` with setup steps and what to verify in the simulator.
- Solutions target iOS 16+ where noted (Chapter 7 navigation). Chapter 14 requires iOS 17+ for SwiftData.
