Chapter 1 Solution: Campus Welcome

Xcode project name: CampusWelcome

Chapter 1 Solution: Campus Welcome

Setup

1. Create a new Multiplatform App in Xcode named CampusWelcome.

2. Replace the default app and content files with the Swift files in this folder.

3. Run the preview and the iOS simulator.

What You Should See

  • A blue "Campus Welcome" title
  • A gray border around the padded content
  • Your name in the subtitle (replace "Your Name")

Files

  • CampusWelcomeApp.swift — app entry point with @main
  • ContentView.swift — welcome screen

Solution Code

Copy these files into your Xcode project. Create a new App project named CampusWelcome, then add each file.

CampusWelcomeApp.swift

import SwiftUI

@main
struct CampusWelcomeApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}

ContentView.swift

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack(spacing: 12) {
            Text("Campus Welcome")
                .font(.largeTitle)
                .foregroundColor(.blue)

            Text("Built by Your Name")
                .font(.headline)
                .foregroundColor(.secondary)
        }
        .padding()
        .border(Color.gray, width: 1)
    }
}

#Preview {
    ContentView()
}
